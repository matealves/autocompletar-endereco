<div id="visao">
<h1>Visão geral</h1>
 Muito comum na maioria dos sites da web, ao preencher o campo CEP de um formulário de endereço, alguns dados se preenchem <strong>automaticamente</strong>, ou seja, o sistema faz uma requisição em uma API usando o CEP que o usuário digitou e tem como retorno os dados do endereço relacionado aquele CEP.
 Layout desenvolvido com <strong>Bootstrap</strong>, com animações suaves e uso de modal, para interação com o usuário...
